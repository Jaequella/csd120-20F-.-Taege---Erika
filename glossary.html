Glossary

Head, Header and Heading
there is only 1 head per html doc, header contains intro content, heading is for heading text.
Navigation(nav)
the nav is to use navigation links like the 3 above for glossary index and links.
P, EM and Blockquote
p is for paragraphs, multple p's can make block-level writing. em puts objects side by side, almost like the opposite.Blockquote is for large chunks of text.
Description,Unordered and Ordered Lists
Description lists are like this one you're reading right now. Unordered lists (UL) are listed with points next to it, Ordered lists (OL) are given numbers beside instead.
Emphasize and Strong
emphasize (em) is used to indicate emphasized text(not to italicize) and (strong) is used to highlight important text.
Cite
used to mark citations to sites info was used from.
Main
Main is unique from page-to-page, often a child of the Body Element
